# Folding-Tile-Illusions-Image-Rotator
Software for rotating images created using the Folding Tile Illusions method

![alt text](https://github.com/ebcrivellaro/Folding-Tile-Illusions-Image-Rotator/blob/main/legacy.png)&rarr;&rarr;&rarr;&rarr;&rarr;![alt text](https://github.com/ebcrivellaro/Folding-Tile-Illusions-Image-Rotator/blob/main/legacy_rot_3x4.png)

### During my undergraduate thesis research, I worked on creating multiple [Folding Tile Illusions](https://diffusionillusion.github.io) using digital painting; this software was created to rotate these paintings in file format.

This Python code allows you to create a rotated version of images created with Folding Tile Illusions. Simply place the image in a folder with the image format `entrada3x3/entrada3x4/entrada4x3/entrada4x4` and have a separate folder named `saida`. Both folders must be in the same directory as the Python file, then just run the application.
